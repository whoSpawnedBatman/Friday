# PRD.md

**Project:** AI Life Operating System (Codename: FRIDAY)\
**Version:** 1.0

------------------------------------------------------------------------

# Product Overview

An AI-first web application that serves as a personal operating system.
Unlike traditional productivity tools, it actively understands the
user's goals, projects, habits, schedule, and context to recommend the
highest-impact next actions.

------------------------------------------------------------------------

# Objectives

## Primary Goals

-   Centralize every aspect of a user's life.
-   Reduce decision fatigue.
-   Increase consistency.
-   Provide proactive AI assistance.
-   Replace multiple productivity apps.

## Non-Goals

-   Social media platform
-   Generic note-taking app
-   Another habit tracker

# Definition of Success

A user should be able to open the application and immediately
understand: 1. What matters today. 2. What to do next. 3. Why it
matters. 4. How it contributes to long-term goals.

If the product consistently answers those four questions, it is
fulfilling its purpose.
